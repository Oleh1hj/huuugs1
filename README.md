# 💚 Huugs — Dating App

> *два атоми — одна орбіта*

Мобільний застосунок знайомств з підтримкою 🇺🇦 / 🇧🇾 мов та авторизацією через Telegram і Instagram.

---

## 🚀 Деплой за 5 хвилин

### 1. Завантаж на GitHub

```bash
# Ініціалізуй git
git init
git add .
git commit -m "🌱 init: Huugs dating app"

# Створи репозиторій на github.com, потім:
git remote add origin https://github.com/YOUR_USERNAME/huugs.git
git branch -M main
git push -u origin main
```

### 2. Деплой на Vercel

1. Зайди на [vercel.com](https://vercel.com) → **Sign up with GitHub**
2. Натисни **"Add New Project"**
3. Вибери репозиторій `huugs`
4. Налаштування залишай за замовчуванням (Vite автоматично визначається)
5. Натисни **"Deploy"** 🎉

Vercel сам визначить що це Vite + React проєкт.

---

## 💻 Локальна розробка

```bash
npm install
npm run dev
```

Відкрий [http://localhost:5173](http://localhost:5173)
